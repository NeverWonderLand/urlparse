> # URLPARSE
>> ## Reads a list of URLs from a file, extracts the domain from each URL, and prints it.

> 1. clone
```
git clone https://github.com/NeverWonderLand/urlparse
```
> 2. requirements
```
pip3 install -r requirements.txt
```
> 3. put you URLs in urls.txt
```
nano urls.txt
```
> 4. run the tool
```
python3 urlparse.py
```
