# URLPARSE
> Reads a list of URLs from a file, extracts the domain from each URL, removes duplicates and prints the list of domains.
<hr>

1. clone
```
git clone https://github.com/NeverWonderLand/urlparse.git
```
2. put your URLs in urls.txt
```
nano urls.txt
```
3. run the tool
```
python3 urlparse.py
```
